\textbf{公理1（原点公理）：}数学操作的原点不是一个标量0，而是一个三维方向向量
\[
\delta_0 = (\delta_x, \delta_y, \delta_z)
\]
其中$\delta_x, \delta_y, \delta_z$不全为零。推论：没有绝对为零的操作起点。每一次加法都携带了一个不可归零的方向偏好。

\textbf{公理2（减法不可清零公理）：}对于任意偏元$a$，
\[
a - a = \varepsilon \neq 0
\]
其中$\varepsilon = (\varepsilon_x, \varepsilon_y, \varepsilon_z)$是一个非零的三维方向向量。推论：减法不是加法的逆操作。加法与减法之间的关系是非对称的、不可逆的呼吸相位关系——就像呼吸的“吸”与“呼”，不是互为逆操作，而是同一周期的两个共轭相位。

\subsection{偏元向量加法}

\textbf{定义1（偏元向量加法）：}对于两个偏元向量$v$和$u$，
\[
v \oplus u = w + \varepsilon
\]
其中$w$是向量和$v+u$在标准意义下的结果，$\varepsilon = (\varepsilon_x, \varepsilon_y, \varepsilon_z)$是加法操作产生的固有偏移向量。加法不是闭合的——每一次加法都会产生一个不可清零的小偏移。

\textbf{定理1（累积方向场定理）：}经过$N$次偏元加法后，累积结果为
\[
P(N) = \left( \sum_{i=1}^N \varepsilon_{x,i},\; \sum_{i=1}^N \varepsilon_{y,i},\; \sum_{i=1}^N \varepsilon_{z,i} \right)
\]
这是一个三维空间中的矢量场，不是一条线。证明：由公理1直接推导。

\subsection{偏元数学的自洽性}

\textbf{定理2（内洽性定理）：}偏元数学的六条核心推论——原点方向性、减法非逆性、加法非闭合性、累积非线形性、方向偏好自旋性、呼吸相位共轭性——全部可以从公理1和公理2唯一推导，且所有推论之间不存在逻辑矛盾。

证明思路：每条推论均由公理直接推出，不依赖外部假设。任何两条推论之间的联合约束均可在偏元加法空间中被同时满足。如果存在一个有限操作使得$a - a = 0$严格成立，则偏元数学的自洽性自动瓦解——该条件本身就是可证伪的。

\subsection{几何滤波算子$F$}

在三维空间中，定义几何滤波算子$F$：
\[
F[P(N)] = \int \frac{1}{r^2(\theta, \varphi)} \cdot M(\theta, \varphi) \cdot P(r) \, dr

其中：
\begin{itemize}
\item $r(\theta, \varphi)$是球壳在方向$(\theta, \varphi)$上的径向坐标；
\item $M(\theta, \varphi)$是方向投影矩阵。
\end{itemize}

\textbf{定理3（球壳径向投影定理）：}在球壳曲率下的径向投影，滤波算子$F$展开至一阶的形式为：
\[
F \approx P_0 \cdot \ln\left(\frac{R_{\text{mean}}}{R_0}\right) \cdot M(\theta, \varphi)
\]
主项的对数形式来自球壳的正曲率径向投影。该式中未预设球壳的形状为正圆、椭球或其他——球壳的精确几何形状应从偏元向量加法方向场的自旋结构中自然长出，而非在推导前预先指定。高阶修正项的形式，取决于长出的几何是否为椭球壳、鞍面或其他形状，本文暂以“高阶修正项”泛指，不预设其具体形式。
